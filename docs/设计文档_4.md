# 增量设计说明（侧边栏可隐藏与布局调整）

本次修改目标：
- 将“开始新对局”按钮移动到“虚着/悔棋/认负”三按钮的正下方。
- 右侧设置边栏改为可隐藏。
- 将提示信息与状态预览移动到右侧设置边栏中。

## 变更点

1) UI 布局调整
- 左侧列：
  - 保持棋盘画布与操作按钮。
  - 按钮排列为两行：第一行「虚着（围棋）/悔棋/认负」，第二行单独放置「开始新对局」以满足位置要求。
  - 存档/读档一行保持不变。
- 右侧列：
  - 使用 Gradio Accordion 容器实现“可隐藏/展开”的设置边栏（标题：设置与提示）。
  - 将「游戏类型/棋盘大小/贴目/主题」等控件放入该侧边栏。
  - 将提示信息与状态预览（status Image）移动到侧边栏内部，提示文案与第二版一致。

2) 交互与事件
- “开始新对局”依旧从右侧控件读取参数，但按钮位于左侧，点击逻辑不变：
  - 先 ctl.set_theme，再 ctl.new_game，修复主题需要点两次的问题仍然有效。
- 其他按钮（虚着、悔棋、认负、保存、读取）行为与此前版本一致。
- 页面加载时，默认初始化 19×19 围棋盘保持不变。

## 影响文件
- ui/app_ui.py：更新布局至两列，右侧为 Accordion，可折叠；移动 status 与提示说明至侧边栏；移动“开始新对局”按钮位置。

## 验收要点
- 右侧设置边栏可通过折叠按钮隐藏和重新显示。
- 提示信息与状态预览显示在侧边栏内部。
- “开始新对局”按钮位于“虚着”“悔棋”“认负”三按钮正下方。
- 主题切换后，首次点击“开始新对局”即生效；默认加载 19×19 围棋盘；弹窗中文、图片文字英文。
